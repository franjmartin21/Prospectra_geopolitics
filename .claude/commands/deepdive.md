Deliver a deep-dive lecture on the topic provided as the argument (e.g. `/deepdive dollar hegemony`).

Steps:
1. Treat the argument after `/deepdive` as the topic. If no topic is given, ask the user what they'd like to explore.
2. Structure the lecture in four parts:
   - **Historical roots**: where did this dynamic come from?
   - **Current state**: what are the key actors, tensions, and incentives today?
   - **Investment framework**: how should an investor think about risk and opportunity in this area? Include specific asset classes, instruments, or geographic exposures.
   - **Scenario analysis**: lay out 2-3 plausible futures for this topic (base case, tail risk, upside) and what each means for portfolios.
3. Flag relevant data sources and a Databricks pipeline idea that could operationalize this analysis.
4. End with 3 pointed questions that would deepen understanding if researched further.
5. Save output to `reports/market_analysis/` with filename `YYYY-MM-DD_<topic-slug>.md`.

This is a graduate-level seminar. Be rigorous, cite thinkers (Kissinger, Mearsheimer, Dalio, etc.) where relevant, and don't shy away from controversial or non-consensus views.
